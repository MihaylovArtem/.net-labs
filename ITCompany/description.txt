IT-компания


Интерфейсы: 

1) IComputerFactory - интерфейс абстрактной фабрики

2) IEmployee - интерфейс сотрудника

3) IComputer - интерфейс компьютера


Классы:

1) Computer - базовый класс компьютера
2) Employee - базовый класс сотрудника

3) Manager - класс сотрудника - менеджера

4) Developer - класс сотрудника - разработчика
5) PC - персональный компьютер c операционной системой Windows

6) Mac - компьютер фирмы Apple и с ОС MacOS
7) MacFactory - конкретная фабрика Маков

8) PCFactory - кокретная фабрика ПК
